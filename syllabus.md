# SWE 402 - Game Programming Syllabus

## Course Overview
This course introduces modern game development principles and interactive media trends. It covers advanced real-time graphics, physics simulation, artificial intelligence, networking, and performance optimization. The course integrates theory, practical programming, and project-based learning to prepare students for real-world game development careers.

## Teaching Method
Prep before lecture (watch videos or read material), then build and iterate during lecture time.

## Prerequisites
- ICS 108: Object-Oriented Programming

## Catalog Description
Introduction to games and interactive media; core game programming principles; modern game development engines and tools; advanced real-time graphics and rendering; physics simulation and collision detection; artificial intelligence and procedural content generation; multiplayer and networked game development; emerging technologies and immersive experiences; performance optimization and debugging techniques; game production and industry practices; game development project.

## Main Objectives
1. Introduce students to modern game development principles and interactive media trends, emphasizing both technical foundations and creative design.
2. Develop practical programming skills for designing, implementing, and optimizing real-time interactive applications across various game genres.
3. Enable students to explore and apply advanced techniques in graphics rendering, physics simulation, artificial intelligence, and procedural content generation.
4. Foster collaborative, project-based learning through the design, development, and deployment of a complete interactive game project.
5. Prepare students for industry practices by integrating agile methodologies, performance optimization, and emerging immersive technologies.

## Course Learning Outcomes
1. Explain key modern game development paradigms, including interactive media trends, advanced graphics, physics simulation, artificial intelligence, and emerging immersive technologies.
2. Apply state-of-the-art programming techniques and engine-agnostic development practices to design, implement, optimize, and rigorously test interactive applications across diverse game genres.
3. Integrate networking and multiplayer architectures to develop scalable, synchronized, and engaging online interactive experiences.
4. Embrace collaborative, iterative project development and ethical industry practices, leveraging agile methodologies and performance optimization to enhance user experience and innovation in game design.

## Topics and Contact Hours
Total contact hours: 45

- Introduction to Games and Interactive Media
- Core Game Programming Principles
- Modern Game Development Engines and Tools
- Advanced Real-Time Graphics and Rendering
- Physics Simulation and Collision Detection
- Artificial Intelligence and Procedural Content Generation
- Multiplayer and Networked Game Development
- Emerging Technologies and Immersive Experiences
- Performance Optimization and Debugging Techniques
- Production and Industry Practices
- Game Development Project

## Assessment Criteria
- Quizzes: 10%
- Assignments: 40% (4 indie games, increasing in complexity)
- Project: 20%
- Final Exam: 30%

## Project
One group project: build a larger game in teams of 3, culminating in a game jam at the end of the semester.
